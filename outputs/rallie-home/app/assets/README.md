# Rallie Image Assets

Replace these files to customize the app graphics:

- `rallie_logo.png`: header logo/mark next to the Rallie wordmark.
- `report_confirmation.jpg`: confirmation-page celebration artwork.

Recommended formats:

- Use `SVG` for logos and line illustrations when possible.
- Use transparent `PNG` or `WebP` for raster artwork.
- Keep the same filenames unless you also update the `src` paths in the page modules.

Recommended sizing:

- `rallie_logo.png`: square artwork, ideally `256x256` or larger.
- `report_confirmation.jpg`: wide artwork, around `640x380`.
- Keep file sizes under `250 KB` for fast mobile loading.
